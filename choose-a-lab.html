Game Design and Development Decal

Choose-A-Lab



Blend Trees

In this lab, you’ll create smooth transitions between 2D animations using Blend Trees. Blend trees are a must for organizing complicated animation transitions in Animator. They’re used to quickly handle transitions between animations when the player provides new input. In 3D games, they actually interpolate between the two animations, creating a smooth transition. We will only be going over the 2D version, but all the concepts apply to 3D animations.

Animations

In this lab, you will be getting a preview of the game animation pipeline, from a Photoshop sprite to a finished Unity animation. Along with introducing the basics of animation principles, we will be introducing two methods of creating animations: a sprite-based method, and a puppet-based method that uses many of the tools that 3D animations use.

Particle Effects

Learn how to make cool 2D special effects (fairly easily extended into 3D)
