Game Design and Development Decal

Spring 2018

Choose-A-Lab



Blend Trees

Artist Friendly!

In this lab, you'll create smooth transitions between 2D animations using Blend Trees. Blend trees are a must for organizing complicated animation transitions in Animator. They're used to quickly handle transitions between animations when the player provides new input. In 3D games, they actually interpolate between the two animations, creating a smooth transition. We will only be going over the 2D version, but all the concepts apply to 3D animations.

Animators

Artist Friendly!

A lab to get you acquainted with Unity's animator and will focus on making animations based on sprite sheets. If you plan to do both the animator and animation lab, it is recommended you do this one first as it is a simpler and less in-depth approach.

Animations

Artist Friendly!

In this lab, you will be getting a preview of the game animation pipeline, from a Photoshop sprite to a finished Unity animation. Along with introducing the basics of animation principles, we will be introducing two methods of creating animations: a sprite-based method, and a puppet-based method that uses many of the tools that 3D animations use.


Particle Effects

Artist Friendly!

Learn how to make cool 2D special effects (fairly easily extended into 3D)

Save Data and Scenes

This lab covers a variety of topics related to scenes, including how to switch between scenes, resetting/reloading a scene, and pausing a scene. It also covers how to save data, both between scenes and after the game has been closed, as well as loading the save data.

UI

Artist Friendly!

A lab to get you acquainted with UI in Unity. You will create a shop using all UI elements. Covers both the scripting required for events and the design behind UI.


Rigid Bodies

The RigidBody, input and movement lab covers the basics of the RigidBody component and how to utilize Unity's physics engine. This lab focuses on manipulating and playing with Unity's physics to be able to see how to get desired effects

Colliders

Learn how to use physical colliders to create an isGrounded check and trigger colliders to create an attack range for a boss.

Audio

Learn how to play sounds in unity, apply filters, and route audio sources through a main soundboard.

Shaders

More Challenging!

In this lab, we will go over the science behind shaders and try out some different shader effects. Material effects, layer effects, and post processing effects are all things that are achieved with shaders. These can be applied to both 2D and 3D games, and the effects that you can achieve are boundless.
